### 🚀 What I'm building

**[Synorva](https://github.com/sup4nova/Synorva)** - A multimodal AI that reads the emotion of an image and generates a matching audio track (valence/arousal model, OpenCV, FastAPI + Vue 3).

**[JobScraper](https://github.com/sup4nova/JobScraper)** - A full-stack job-discovery app: a FastAPI backend orchestrating resilient scrapers, a Discord bot, and a Tinder-style Vue swipe UI, with AI-assisted CV generation.

**[sup4nova.com](https://sup4nova.com)** - My self-built Vue 3 portfolio, hosted on OVH.

**[JavaCryptoTool](https://github.com/sup4nova/JavaCryptoTool)** - AES encryption/decryption tool with a web interface and Java servlet backend (school project).

🚧 **In progress - [RandomSpotify](https://sup4nova.com)** - Real-time multiplayer Spotify party game (FastAPI + WebSockets, OAuth 2.0 PKCE, Vue SPA).
